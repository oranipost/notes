### Difference Quotient

*   m = [f(x2) - f(x1)] / (x2 - x1)
      = [f(x + h) - f(x)] / [(x + h) - x]
      = [f(x + h) - f(x)] / h
    
    *   Note: the m is the slope of the secant line, the point are (x, f(x)),
        another point is (x+h, f(x+h)), they are both on the same curve. The secant 
        line is pass through the two points.
    
    *   Tips: Generally, the factor x is usually elinminated, thus the result do not 
        have x, only the h remained


*   Take the Example: f(x) = 2x + 1, g(x) = x + 1, 
    f(g(x)): the result will be 2(x + 1) + 1 = 2x + 3
    (f + g) (x): the result will be (2x + 1) + (x + 1) 
    (f * g) (x): the result will be (2x + 1) * (x + 1) 
    (f / g) (x): the result will be (2x + 1) / (x + 1)
    (f - g) (x): the result will be (2x + 1) - (x + 1)
    (f ∘ g) (0): the result will be f(g(0))
    (g ∘ f) (0): the result will be g(f(0))
    
    *   Note: the ∘ means "of"

*   If h(x) = (f ∘ g) (x) & h(x) = |2(x^2) - 5| 

    *   We can make the h(x) into g(x) = 2(x^2) - 5, f(x) = |x|
    *   Thus, (f ∘ g) (x) = f(g(x)) = |g(x)| = |2(x^2) - 5|
